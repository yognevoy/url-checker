# url-checker

A command-line tool that checks the status of URLs.

## Overview

url-checker reads a file containing URLs and checks each one, reporting the HTTP status code and latency.

## Usage

```bash
url-checker [-o output.txt] <urls_file>
```

## Flags

| Flag | Default | Description          |
|------|---------|----------------------|
| `-o` |         | Output file path     |
